01Why This Unit Matters

Thermochemistry answers the question every chemist asks first: does this reaction release or absorb energy, and will it happen on its own? Enthalpy and entropy appear in every JEE paper — sometimes as direct numericals, sometimes hidden in equilibrium (ΔG° = −RT ln K) or electrochemistry (ΔG° = −nFE°).

📊 JEE Pattern: Numericals + Concept Questions
Typical C4 questions: Hess's law calculation (1–2 marks), predict sign of ΔS (1 mark), find temperature for spontaneity (1 mark), ΔH vs ΔU calculation (1 mark). Total: 4–5 reliable marks with focused preparation.
✅ Bridge to Equilibrium
ΔG° = −RT ln K connects thermochemistry directly to Chemical Equilibrium (C5). A solid understanding of Gibbs energy makes C5 much easier to approach.

02System, Surroundings & Sign Conventions

System Types
  • Open: exchanges both matter and energy (e.g. open beaker)
  • Closed: exchanges only energy, not matter (e.g. sealed flask)
  • Isolated: exchanges neither (e.g. thermos flask — ideal)
Sign Conventions (IUPAC)
  • q > 0: heat flows INTO system (endothermic)
  • q < 0: heat flows OUT of system (exothermic)
  • w > 0: work done ON system (compression)
  • w < 0: work done BY system (expansion)
  • Work at constant P: w = −PΔV
Process Types
  • Isothermal: constant T; ΔU = 0 for ideal gas
  • Adiabatic: q = 0 (no heat exchange)
  • Isobaric: constant P; heat = ΔH
  • Isochoric: constant V; heat = ΔU
⚠ The JEE Sign Trap
Work done BY the gas is negative in the IUPAC convention (w = −PΔV). When gas expands, ΔV > 0, so w < 0 — the system loses energy doing work on surroundings. Many textbooks use the opposite sign (w = +PΔV) — always check which convention is being used.

03First Law of Thermodynamics

First Law: Energy is Conserved

ΔU = q + w

ΔU = change in internal energy  |  q = heat  |  w = work

At constant volume (ΔV = 0): w = 0, so ΔU = qv (heat at constant volume = ΔU)

At constant pressure: qp = ΔH (heat at constant pressure = enthalpy change)

💡 State Functions vs Path Functions
State functions depend only on initial and final states, not the path: ΔU, ΔH, ΔS, ΔG.
Path functions depend on the path taken: q (heat) and w (work).
This is why Hess's Law works — ΔH is a state function, so the path doesn't matter.

04Enthalpy — ΔH vs ΔU

The Key Relationship

ΔH = ΔU + ΔngRT

Δng = moles of gaseous products − moles of gaseous reactants

This formula corrects for the PΔV work done by/on the gas phase during a reaction at constant pressure. Liquids and solids contribute negligible volume change.

ΔngMeaningRelationshipExample
Δng = 0Equal moles of gas on both sidesΔH = ΔUH₂(g) + Cl₂(g) → 2HCl(g)
Δng > 0More gas producedΔH > ΔUCaCO₃(s) → CaO(s) + CO₂(g); Δng=1
Δng < 0Less gas producedΔH < ΔUN₂ + 3H₂ → 2NH₃; Δng=−2

💡 Standard State Convention
Standard state = most stable form of element at 298 K and 1 atm. C(graphite), H₂(g), O₂(g), N₂(g), Na(s), etc. Carbon standard state is graphite, NOT diamond — so ΔH°f of graphite = 0 but ΔH°f of diamond ≠ 0.

06Hess's Law — Step-by-Step Method

Because enthalpy is a state function, the total ΔH of a reaction is the same regardless of how many steps it takes. You can combine any set of reactions algebraically to derive the ΔH for a target reaction.

Hess's Law — Definitive Method
  1. Write the target equation clearly at the top.
  2. Identify which given equations contribute each species in the target.
  3. Reverse equations if needed (flip sign of ΔH).
  4. Multiply equations to match stoichiometry (multiply ΔH by same factor).
  5. Add all equations — cancel species appearing on both sides.
  6. Add all ΔH values to get total ΔH.
Worked: ΔH°f of CO using Hess's Law

Target: C(s) + ½O₂(g) → CO(g)   ΔH = ?

Given:

  • Eq 1: C(s) + O₂(g) → CO₂(g)   ΔH₁ = −393 kJ
  • Eq 2: CO(g) + ½O₂(g) → CO₂(g)   ΔH₂ = −283 kJ

Step 1: Reverse Eq 2 (to get CO on product side): CO₂(g) → CO(g) + ½O₂(g), ΔH = +283 kJ

Step 2: Add Eq 1 + reversed Eq 2:

C(s) + O₂(g) + CO₂(g) → CO₂(g) + CO(g) + ½O₂(g)

Step 3: Cancel CO₂ and ½O₂: C(s) + ½O₂(g) → CO(g)

ΔH = −393 + 283 = −110 kJ

Formation Enthalpy Shortcut

ΔH°rxn = Σ ΔH°f(products) − Σ ΔH°f(reactants)

This is Hess's law applied to formation enthalpies. Remember: ΔH°f of any element in its standard state = 0.

07Bond Enthalpy Method

When ΔH°f values aren't given, use bond enthalpies (also called bond dissociation energies):

Bond Enthalpy Formula

ΔH°rxn = Σ(bond enthalpies of bonds broken) − Σ(bond enthalpies of bonds formed)

Breaking bonds = endothermic (+)  |  Forming bonds = exothermic (−)

Worked: H₂ + Cl₂ → 2HCl

Bonds broken: 1 H–H (436 kJ) + 1 Cl–Cl (242 kJ) = 678 kJ

Bonds formed: 2 H–Cl (2 × 431 = 862 kJ)

ΔH = 678 − 862 = −184 kJ (exothermic ✓)

⚠ Bond Enthalpy Method is Approximate
Bond enthalpies are average values — the actual bond energy varies slightly depending on the molecule. Results from this method are approximate. When ΔH°f values are available, use the Hess's law/formation method — it gives exact answers.

08Entropy — Predicting ΔS

Entropy (S) is a measure of disorder/randomness. The Second Law states: the entropy of the universe increases for any spontaneous process.

Rules for Predicting Sign of ΔS
ΔS > 0 (Entropy Increases)
  • Solid → Liquid → Gas (phase changes)
  • Dissolving a solid in a liquid
  • Reaction produces more moles of gas
  • Increasing temperature
  • Mixing of gases
ΔS < 0 (Entropy Decreases)
  • Gas → Liquid → Solid (phase changes)
  • Gas dissolves into liquid
  • Reaction produces fewer moles of gas
  • Formation of precipitate from solution
  • N₂ + 3H₂ → 2NH₃ (4 mol gas → 2 mol gas)
💡 Absolute Entropy
Unlike ΔH and ΔG, absolute entropies S° can be measured (Third Law: S = 0 for perfect crystal at 0 K). Standard entropy values are tabulated. ΔS°rxn = Σ S°(products) − Σ S°(reactants). Note: ΔS°f of elements ≠ 0 (unlike ΔH°f).

09Gibbs Free Energy & Spontaneity

Gibbs Free Energy

ΔG = ΔH − TΔS

  • ΔG < 0: Spontaneous (favourable) — reaction proceeds without external energy
  • ΔG > 0: Non-spontaneous (unfavourable) — needs external energy input
  • ΔG = 0: System is at equilibrium

10Spontaneity Matrix

The combination of ΔH and ΔS signs determines when a reaction is spontaneous. This table must be memorised — it appears in JEE almost every year.

💡 Crossover Temperature
For the borderline cases (ΔH and ΔS both −, or both +), the crossover temperature where ΔG = 0 is:

Tcrossover = ΔH / ΔS

(with ΔH in J/mol and ΔS in J/mol·K, or both in kJ). Reaction is spontaneous below this T if ΔH < 0, ΔS < 0; above this T if ΔH > 0, ΔS > 0.

11ΔG° and Equilibrium Constant K

The Bridge Between Thermodynamics and Equilibrium

ΔG° = −RT ln K

R = 8.314 J/mol·K  |  T in Kelvin  |  K = equilibrium constant

K valueln KΔG°Meaning
K >> 1Positive, largeNegative, largeProducts strongly favoured
K = 100Equal tendency for products and reactants
K << 1Negative, largePositive, largeReactants strongly favoured
💡 Also useful: ΔG = ΔG° + RT ln Q
This generalised form gives ΔG at any composition (not just standard). At equilibrium Q = K → ΔG = ΔG° + RT ln K = −RT ln K + RT ln K = 0. Makes sense — ΔG = 0 at equilibrium.

13Worked Examples

Example 1 — ΔH vs ΔU (Easy)

Q: For N₂(g) + 3H₂(g) → 2NH₃(g), ΔH = −92 kJ at 300 K. Find ΔU.

Step 1: Δng = moles of gas produced − consumed = 2 − (1+3) = −2

Step 2: ΔH = ΔU + ΔngRT → ΔU = ΔH − ΔngRT

Step 3: ΔU = −92000 − (−2)(8.314)(300) = −92000 + 4988 = −87012 J ≈ −87 kJ

Example 2 — Hess's Law (Medium)

Q: Find ΔH°c for benzene (C₆H₆, l). Given: ΔH°f(CO₂) = −393, ΔH°f(H₂O,l) = −286, ΔH°f(C₆H₆,l) = +49 kJ/mol

Target: C₆H₆(l) + 15/2 O₂(g) → 6CO₂(g) + 3H₂O(l)

ΔH°c = [6(−393) + 3(−286)] − [49 + 0] = [−2358 − 858] − 49 = −3265 kJ/mol

Example 3 — Spontaneity Temperature (Medium)

Q: For a reaction: ΔH = +30 kJ, ΔS = +100 J/K. At what minimum temperature is it spontaneous?

Step 1: Need ΔG < 0, i.e., ΔH − TΔS < 0

Step 2: 30000 − T × 100 < 0 → T > 30000/100 = 300 K

Spontaneous above 300 K (ΔH > 0, ΔS > 0 → favourable at high T).

Example 4 — Bond Energy (Hard)

Q: Find ΔH for CH₄(g) + 2O₂(g) → CO₂(g) + 2H₂O(g). Bond energies: C–H=413, O=O=498, C=O=799, O–H=463 kJ/mol.

Bonds broken: 4 × C–H + 2 × O=O = 4(413) + 2(498) = 1652 + 996 = 2648 kJ

Bonds formed: 2 × C=O (in CO₂) + 4 × O–H (in 2H₂O) = 2(799) + 4(463) = 1598 + 1852 = 3450 kJ

ΔH = 2648 − 3450 = −802 kJ (exothermic ✓)

14JEE Previous-Year Questions

📋 Most Tested Thermochemistry Patterns
  1. Calculate ΔH or ΔU using ΔngRT correction
  2. Hess's law: combine 2–3 equations to find ΔH for target reaction
  3. Predict spontaneity from ΔH/ΔS signs and temperature
  4. Find crossover temperature where reaction becomes spontaneous
  5. Predict sign of ΔS for a given physical/chemical change
  6. Connect ΔG° to K (larger K → more negative ΔG°)
PYQ — JEE Main 2023

Q: Standard enthalpy of formation of CH₄(g), CO₂(g) and H₂O(l) are −74.8, −393.5 and −285.5 kJ/mol. The standard enthalpy of combustion of methane is:

CH₄ + 2O₂ → CO₂ + 2H₂O

ΔH°c = [−393.5 + 2(−285.5)] − [−74.8 + 0] = [−393.5 − 571] + 74.8 = −889.7 kJ/mol

PYQ — JEE Main 2022

Q: For a reaction at 300 K: ΔH° = −57.32 kJ/mol, ΔS° = −176.04 J/mol·K. The correct statement about the reaction is:

ΔG° = ΔH° − TΔS° = −57320 − 300(−176.04) = −57320 + 52812 = −4508 J/mol ≈ −4.5 kJ/mol

ΔG° < 0 → spontaneous at 300 K. (ΔH < 0, ΔS < 0 → spontaneous only at low T — and 300 K is below crossover T = 57320/176.04 ≈ 325 K)

15Common Traps & Mistakes

❌ Trap 1: Mixing up ΔH and ΔU sign corrections
ΔH = ΔU + ΔngRT — the Δng applies only to gaseous species. Liquids and solids don't contribute to Δng. For C + O₂ → CO₂, Δng = 1 − 1 = 0 (not counting the solid C).
❌ Trap 2: ΔH°f of elements ≠ 0 for entropy
Standard formation enthalpy of elements = 0. But standard entropy S° of elements ≠ 0. Don't confuse them. S°(H₂,g) = 130.6 J/mol·K (not zero).
❌ Trap 3: Units mismatch in ΔG = ΔH − TΔS
ΔH is usually given in kJ/mol, ΔS in J/mol·K. Convert one to match the other before calculating ΔG. Common error: −100 kJ − 298 × 200 J/K → off by 1000×.
❌ Trap 4: Catalyst changes ΔG or K
Catalyst does NOT change ΔG, ΔH, ΔS, or K. It only lowers activation energy — the reaction reaches equilibrium faster. The thermodynamic outcome is unchanged.
❌ Trap 5: Spontaneous ≠ fast
A reaction can be spontaneous (ΔG < 0) but extremely slow. Diamond spontaneously converts to graphite at room temperature — but effectively never does because the kinetic barrier is enormous.

16One-Page Cheat Card

🔥 Thermochemistry — Rapid Reference

Core Equations

  • ΔU = q + w  |  w = −PΔV
  • ΔH = ΔU + ΔngRT
  • ΔH°rxn = ΣΔH°f(P) − ΣΔH°f(R)
  • ΔG = ΔH − TΔS
  • ΔG° = −RT ln K
  • Tcross = ΔH/ΔS

Spontaneity Matrix

  • −ΔH, +ΔS → always spontaneous ✅
  • +ΔH, −ΔS → never spontaneous ❌
  • −ΔH, −ΔS → spontaneous at low T
  • +ΔH, +ΔS → spontaneous at high T

ΔS Signs

  • +ΔS: solid→gas, more gas moles, dissolving
  • −ΔS: gas→solid, fewer gas moles, precipitate

Key ΔH°f Values

  • CO₂(g): −393 kJ/mol
  • H₂O(l): −286 kJ/mol
  • H₂O(g): −242 kJ/mol
  • Any element (std): 0
