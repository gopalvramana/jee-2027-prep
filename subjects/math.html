M1 Trigonometry March 2026 — 4 weeks 8–10% · 13 sessions ▼

Unit Goal: Score 8–10 marks out of the ~10% allocated to Trigonometry in JEE. Complete all 13 sessions and score 12+/20 on the final JEE practice set before moving to Quadratics.

T1Trig Ratios, Quadrants & ASTC Rule1.5 hrs▼
Concepts
  • Six trig ratios: sin, cos, tan, cosec, sec, cot — definitions from right triangle
  • ASTC rule: All Positive (Q1), Sin positive (Q2), Tan positive (Q3), Cos positive (Q4)
  • Reference angle: always acute, find trig value then apply correct sign
  • Trig ratios for 0°, 30°, 45°, 60°, 90° — memorise all 6 ratios for each
  • Reciprocal identities: cosec = 1/sin, sec = 1/cos, cot = 1/tan
  • Converting between degrees and radians: π radians = 180°
Tip: Draw the unit circle with all 4 quadrants. Mark which ratios are positive in each. This is the base for everything that follows.
Practice Problems
  1. Find all 6 trig ratios for θ = 120°, 225°, 315°.
  2. If sin θ = 3/5 and θ is in Q2, find cos θ, tan θ.
  3. Find the value of sin 150° + cos 240° + tan 315°.
  4. Convert: 135° to radians, 5π/6 to degrees.
T2Standard Values & Exact Values Table1.5 hrs▼
Concepts
  • sin 0=0, sin 30=1/2, sin 45=1/√2, sin 60=√3/2, sin 90=1
  • cos 0=1, cos 30=√3/2, cos 45=1/√2, cos 60=1/2, cos 90=0
  • tan 0=0, tan 30=1/√3, tan 45=1, tan 60=√3, tan 90=undefined
  • Negative angles: sin(−θ) = −sin θ (odd); cos(−θ) = cos θ (even)
  • Complementary angles: sin(90−θ) = cos θ; tan(90−θ) = cot θ
  • Supplementary: sin(180−θ) = sin θ; cos(180−θ) = −cos θ
Tip: Make a flashcard with the standard values table. Quiz yourself daily until instant recall at 5 angles × 6 ratios = 30 values.
Practice Problems
  1. Without calculator: find sin 210°, cos 300°, tan 135°.
  2. Simplify: sin(180°+θ)·cos(360°−θ)·tan(90°+θ).
  3. Find the value of cos²45° + sin²30° + tan²60°.
  4. If sin A = cos B, find A + B (where A, B are acute angles).
T3Fundamental Identities1.5 hrs▼
Concepts
  • Pythagorean: sin²θ + cos²θ = 1 — the master identity
  • From Pythagorean: 1 + tan²θ = sec²θ and 1 + cot²θ = cosec²θ
  • Using identities to simplify expressions: substitute, factor, reduce
  • Given one ratio → find all others using identities (no quadrant info needed)
  • Common manipulation: sin²θ = 1 − cos²θ = (1−cosθ)(1+cosθ)
Tip: When proving identities, work on one side only. Usually start with the more complex side. Convert everything to sin and cos first.
Practice Problems
  1. If tan θ = 5/12, find sin θ and cos θ without knowing the quadrant.
  2. Prove: (sin θ + cos θ)² + (sin θ − cos θ)² = 2.
  3. Simplify: (sec²θ − 1)(cosec²θ − 1).
  4. Prove: sin⁴θ − cos⁴θ = 2sin²θ − 1.
T4Compound & Double Angle Formulas1.5 hrs▼
Concepts
  • sin(A±B) = sinA cosB ± cosA sinB
  • cos(A±B) = cosA cosB ∓ sinA sinB
  • tan(A±B) = (tanA ± tanB) / (1 ∓ tanA tanB)
  • Double angle: sin 2A = 2 sinA cosA
  • Double angle: cos 2A = cos²A − sin²A = 2cos²A − 1 = 1 − 2sin²A
  • Double angle: tan 2A = 2tanA / (1 − tan²A)
Tip: JEE frequently asks you to find exact values like sin 75°, cos 15°, tan 105° using compound angle formulas. Practice these immediately.
Practice Problems
  1. Find exact value of sin 75° and cos 15° using compound formulas.
  2. If sin A = 3/5 and cos B = 5/13, find sin(A+B).
  3. If tan A = 1/7 and tan B = 1/3, show that 2(A+B) = 45°.
  4. Given sin θ = 1/√5, find sin 2θ and cos 2θ.
T5Triple Angle & Half Angle Formulas1.5 hrs▼
Concepts
  • Triple angle: sin 3A = 3sinA − 4sin³A
  • Triple angle: cos 3A = 4cos³A − 3cosA
  • Half angle: sin A/2 = ±√((1−cosA)/2)
  • Half angle: cos A/2 = ±√((1+cosA)/2)
  • Half angle: tan A/2 = sinA/(1+cosA) = (1−cosA)/sinA
  • Sign of half-angle formulas depends on the quadrant of A/2
Tip: For triple angle: memorise sin 3A = 3sinA − 4sin³A. Derive others from this. Don't memorise all — derive from double + compound.
Practice Problems
  1. Find sin 3 × 20° using triple angle formula.
  2. Find cos A/2 if cos A = 3/5 and A is in Q1.
  3. Prove: (sin 3A / sin A) − (cos 3A / cos A) = 2.
  4. Find tan 22.5° using half-angle formula.
T6Sum-to-Product & Product-to-Sum1.5 hrs▼
Concepts
  • sinC + sinD = 2 sin((C+D)/2) cos((C-D)/2)
  • sinC − sinD = 2 cos((C+D)/2) sin((C-D)/2)
  • cosC + cosD = 2 cos((C+D)/2) cos((C-D)/2)
  • cosC − cosD = −2 sin((C+D)/2) sin((C-D)/2)
  • 2sinA cosB = sin(A+B) + sin(A−B)
  • 2cosA cosB = cos(A−B) + cos(A+B)
Tip: Sum-to-product is used heavily in equation solving and proving identities involving sums of trig functions. Recognise the pattern and apply directly.
Practice Problems
  1. Express sin 70° + sin 20° as a product.
  2. Find the value of cos 20° + cos 100° + cos 140°.
  3. Prove: (sin A + sin 3A) / (cos A + cos 3A) = tan 2A.
  4. Simplify: sin 10° + sin 50° − sin 70°.
T7Review & Mixed Practice (Week 2 End)1.5 hrs▼
Concepts to Consolidate
  • Review all formulas from T1–T6: standard values, compound, double, half, sum-to-product
  • Identify which formula to use given the structure of the problem
  • Mixed JEE-style problems requiring multiple formula applications
  • Proving complex identities combining multiple concepts
Tip: Try 10 JEE past questions from Trigonometry (identities section). Time yourself — target under 3 minutes per question.
Practice Problems
  1. Find the value of sin²10° + sin²50° − sin 10° sin 50°.
  2. Prove: tan 3A − tan 2A − tan A = tan A · tan 2A · tan 3A.
  3. If tan(π/4 + θ) + tan(π/4 − θ) = a, prove that tan²θ = (a−2)/(a+2)... or find cos 2θ.
  4. Simplify: cos²A + cos²(A+60°) + cos²(A−60°).
T8Trigonometric Equations — General Solutions1.5 hrs▼
Concepts
  • General solution of sin θ = a: θ = nπ + (−1)ⁿ arcsin(a), n ∈ Z
  • General solution of cos θ = a: θ = 2nπ ± arccos(a), n ∈ Z
  • General solution of tan θ = a: θ = nπ + arctan(a), n ∈ Z
  • Solve by reducing to single ratio using identities
  • Parametric: equations of form a sinθ + b cosθ = c → use auxiliary angle method
  • Range of solutions: often asked for solutions in [0°, 360°] or [0, 2π]
Tip: Always find the principal value first, then apply the general formula. For solutions in a given interval, list them one by one — don't guess.
Practice Problems
  1. Find general solution of: 2 sin²θ − sin θ − 1 = 0.
  2. Find all solutions of sin 2θ = cos θ in [0°, 360°].
  3. Find the number of solutions of cos 2θ = sin θ in [0°, 360°].
  4. Solve: √3 cos θ + sin θ = 2. Find general solution.
T9Inverse Trigonometry1.5 hrs▼
Concepts
  • arcsin: domain [−1,1], range [−π/2, π/2]
  • arccos: domain [−1,1], range [0, π]
  • arctan: domain ℝ, range (−π/2, π/2)
  • arcsin(x) + arccos(x) = π/2 — important identity
  • arctan(x) + arctan(y) = arctan((x+y)/(1−xy)) for xy < 1
  • Simplification: arcsin(sin θ) = θ only if θ ∈ [−π/2, π/2]
Tip: Ranges of inverse functions must be memorised exactly — JEE questions hinge on whether an angle is in the correct range.
Practice Problems
  1. Find the value of arcsin(1/2) + arccos(1/2).
  2. Simplify: arctan(1/2) + arctan(1/3).
  3. Find the value of arctan(1) + arctan(2) + arctan(3).
  4. Prove: 2 arctan(1/5) + arctan(1/8) = π/4.
T10Properties of Triangles1.5 hrs▼
Concepts
  • Sine rule: a/sinA = b/sinB = c/sinC = 2R (R = circumradius)
  • Cosine rule: a² = b² + c² − 2bc cosA (and cyclic forms)
  • Area of triangle: Δ = ½ ab sinC = ½ bc sinA = ½ ca sinB
  • Heron's formula: Δ = √(s(s−a)(s−b)(s−c)) where s = (a+b+c)/2
  • Inradius: r = Δ/s; Circumradius: R = abc/4Δ
  • Projection formulas: a = b cosC + c cosB
Tip: For any triangle problem — write down which formula connects the given quantities. Sine rule for angle-side pairs; Cosine rule when you have 2 sides and included angle.
Practice Problems
  1. In triangle ABC, a=5, b=6, c=7. Find angle A using cosine rule.
  2. Find the area of triangle with sides 6, 8, 10.
  3. In triangle ABC, if a/(b+c) = b/(a+c), find angle C.
  4. Find circumradius and inradius of equilateral triangle with side 6.

M2 Quadratics & Functions April 2026 6–8% · 6 sessions ▼

Unit Goal: Understand the link between roots and coefficients, master domain/range, and solve quadratic inequalities fluently. Functions form the base for calculus in Phase 2.

Q1Quadratic Equations — Roots & Discriminant1.5 hrs▼
Concepts
  • Standard form: ax² + bx + c = 0; roots by quadratic formula
  • Discriminant Δ = b² − 4ac: Δ>0 real distinct, Δ=0 equal, Δ<0 complex
  • Vieta's formulas: α+β = −b/a, αβ = c/a
  • Form quadratic from roots: x² − (α+β)x + αβ = 0
  • Nature of roots for integer/rational coefficients
Tip: Vieta's formulas let you find α²+β², α³+β³ without solving for roots. Learn to use them — JEE tests this constantly.
Practice Problems
  1. Find the roots and nature for: 2x² − 3x − 5 = 0.
  2. If α, β are roots of x²−5x+6=0, find α²+β² and α³+β³.
  3. Find k if one root of 2x²−5x+k=0 is twice the other.
  4. Form a quadratic with roots 2+√3 and 2−√3.
Q2Quadratic Inequalities & Sign Analysis1.5 hrs▼
Concepts
  • Sign of ax²+bx+c: parabola opens up (a>0) — positive outside roots, negative between
  • Solve ax²+bx+c > 0: find roots, determine sign by parabola direction
  • Wavy curve (number line method) for higher degree inequalities
  • Location of roots: conditions for both roots in (p, q) interval
  • Max/min value: vertex at x = −b/2a; min = c − b²/4a for a>0
Tip: Draw the parabola sketch before solving any quadratic inequality. Direction (a>0 or a<0) determines everything.
Practice Problems
  1. Solve: x² − 5x + 6 > 0.
  2. Solve: 2x² − 3x − 2 ≤ 0.
  3. Find values of k for which x²−kx+4>0 for all real x.
  4. Find the minimum value of 3x² − 6x + 5.
Q3Functions — Domain, Range & Types1.5 hrs▼
Concepts
  • Domain: set of valid inputs (avoid division by zero, square root of negative, log of non-positive)
  • Range: set of all possible outputs — find by solving y = f(x) for x, determine valid y
  • One-one (injective): each output corresponds to exactly one input
  • Onto (surjective): every element of codomain is mapped to
  • Bijective: one-one AND onto — invertible
  • Even function: f(−x) = f(x); Odd: f(−x) = −f(x)
Tip: For domain of √(f(x)): set f(x) ≥ 0. For log(f(x)): set f(x) > 0. Combine conditions using intersection.
Practice Problems
  1. Find domain of f(x) = √(x² − 5x + 6) / (x − 1).
  2. Find range of f(x) = (x−1)/(x+1).
  3. Is f(x) = x/(x²+1) one-one? onto (codomain = ℝ)?
  4. Classify as even, odd, or neither: f(x) = x³ + x, g(x) = x² + 1, h(x) = x + sinx.
Q4Composite & Inverse Functions1.5 hrs▼
Concepts
  • Composite: (fog)(x) = f(g(x)) — apply g first, then f
  • fog ≠ gof in general
  • Inverse f⁻¹ exists iff f is bijective
  • Find f⁻¹: write y=f(x), solve for x in terms of y, swap x↔y
  • f(f⁻¹(x)) = x and f⁻¹(f(x)) = x — definition
  • Graph of f⁻¹ is reflection of f about y = x
Tip: When finding fog, start from the inside. Write out g(x), then substitute every x in f(x) with g(x). Don't try to combine mentally.
Practice Problems
  1. f(x)=2x+3, g(x)=x². Find fog(2) and gof(2).
  2. Find f⁻¹(x) for f(x) = (x−1)/(x+2).
  3. If f(x)=x² (x≥0), find f⁻¹. Verify f(f⁻¹(4)) = 4.
  4. Find domain and range of f⁻¹ when f: [1,3] → [1,9], f(x) = x².
Q5Graphs of Standard Functions1.5 hrs▼
Concepts
  • Know shape, domain, range: y=x, x², x³, √x, 1/x, |x|, eˣ, ln x, sin x, cos x
  • Transformations: f(x)+c (shift up), f(x+c) (shift left), cf(x) (stretch), f(cx) (compress)
  • |f(x)|: reflect the negative part upward
  • f(|x|): reflect the right side to the left
  • Graph of greatest integer function [x] — step function
Tip: Apply transformations in order: inside the function first (horizontal), then outside (vertical). Keep a mental list of 10 standard graphs.
Practice Problems
  1. Sketch: y = |x² − 4|.
  2. From graph of f(x), sketch f(x−2)+3.
  3. Find all integer solutions of [x]² − 3[x] + 2 = 0.
  4. Sketch y = |sin x| for x ∈ [0, 2π].
Q6Complex Numbers & Argand Plane1.5 hrs▼
Concepts
  • i = √(−1); i² = −1, i³ = −i, i⁴ = 1 — cyclic pattern
  • z = a + bi; modulus |z| = √(a²+b²); argument θ = arctan(b/a)
  • Conjugate z̄ = a − bi; zz̄ = |z|²
  • Polar form: z = r(cosθ + i sinθ) = re^(iθ)
  • De Moivre's theorem: (cosθ + i sinθ)ⁿ = cos(nθ) + i sin(nθ)
  • Cube roots of unity: 1, ω, ω² where ω = e^(2πi/3), 1+ω+ω² = 0
Tip: For problems involving ω (cube roots of unity), the key identity is 1+ω+ω²=0. Use this to simplify massive expressions in seconds.
Practice Problems
  1. Find (1+i)⁸ using De Moivre's theorem.
  2. If z = 3 + 4i, find |z|, arg(z), and z⁻¹.
  3. If 1, ω, ω² are cube roots of unity, find (1+ω)⁵+(1+ω²)⁵.
  4. Find all complex numbers z such that |z| = 1 and z² = z̄.

M3 Coordinate Geometry May 2026 8–10% · 5 sessions ▼

Unit Goal: Master straight lines and circles completely. Get comfortable with standard conic forms — parabola is highest yield; ellipse/hyperbola tested less frequently.

G1Straight Lines — Slope, Distance, Angle1.5 hrs▼
Concepts
  • Slope m = (y₂−y₁)/(x₂−x₁); angle with x-axis θ where tanθ = m
  • Forms: slope-intercept y=mx+c, two-point, intercept x/a+y/b=1
  • Angle between two lines: tanθ = |(m₁−m₂)/(1+m₁m₂)|
  • Parallel: m₁=m₂; Perpendicular: m₁m₂ = −1
  • Distance from point (x₁,y₁) to line ax+by+c=0: |ax₁+by₁+c|/√(a²+b²)
  • Foot of perpendicular and reflection formulas
Tip: The distance formula is JEE's most-used coordinate geometry formula. Practice it until it's automatic.
Practice Problems
  1. Find angle between lines y=2x+3 and 3y=x−5.
  2. Find distance from (3,−4) to line 3x−4y+10=0.
  3. Lines 2x+y=4 and x−2y=3. Find their intersection and angle between them.
  4. Reflect point (1, 3) about the line y = x+1.
G2Circles — Equation, Tangent, Chord1.5 hrs▼
Concepts
  • General: (x−h)²+(y−k)² = r² → centre (h,k), radius r
  • Standard expanded: x²+y²+2gx+2fy+c=0 → centre (−g,−f), r = √(g²+f²−c)
  • Tangent at (x₁,y₁): xx₁+yy₁+g(x+x₁)+f(y+y₁)+c=0
  • Condition for tangency: distance from centre = radius
  • Length of tangent from external point: √(x₁²+y₁²+2gx₁+2fy₁+c)
  • Chord of contact: same formula as tangent at point
Tip: To find if a line is tangent to a circle: put the line into the circle, use discriminant = 0. Don't memorise separate tangent condition.
Practice Problems
  1. Circle x²+y²−6x−4y+9=0: find centre, radius, and tangent at (1,2).
  2. Find length of tangent from (5,12) to x²+y²=169.
  3. Two circles x²+y²=4 and x²+y²−6x+5=0. Find common tangents.
  4. Circle through (0,0), (4,0), (0,3): find equation.
G3Parabola — Standard Form, Focus, Directrix1.5 hrs▼
Concepts
  • y² = 4ax: opens right, focus (a,0), directrix x=−a, vertex (0,0)
  • y² = −4ax: opens left; x² = 4ay: opens up; x² = −4ay: opens down
  • Parametric: (at², 2at) for y²=4ax
  • Tangent at (at², 2at): ty = x + at²
  • Normal at parameter t: y = −tx + 2at + at³
  • Focal chord: if one end has parameter t, the other has −1/t
Tip: Identify which standard form: y²=4ax (sideways) or x²=4ay (upright). Write down a, then find focus/directrix directly.
Practice Problems
  1. For y²=12x: find focus, directrix, length of latus rectum.
  2. Find the tangent to y²=8x that is parallel to x−y+3=0.
  3. Normals to y²=4x at points t=1 and t=2 meet at a point. Find it.
  4. Find the equation of parabola with focus (1,0) and directrix x=−1.
G4Ellipse & Hyperbola Basics1.5 hrs▼
Concepts
  • Ellipse x²/a²+y²/b²=1 (a>b): foci (±c,0), c²=a²−b², eccentricity e=c/a < 1
  • Length of major axis=2a, minor=2b, latus rectum=2b²/a
  • Hyperbola x²/a²−y²/b²=1: foci (±c,0), c²=a²+b², e>1
  • Rectangular hyperbola xy=c²: eccentricity √2, asymptotes x=0, y=0
  • Tangent to ellipse: xx₁/a²+yy₁/b²=1
Tip: Remember: for ellipse, c²=a²−b²; for hyperbola, c²=a²+b². The + vs − is the single most common error.
Practice Problems
  1. Ellipse x²/25+y²/9=1: find a, b, c, eccentricity, foci.
  2. Hyperbola x²/16−y²/9=1: find asymptotes and eccentricity.
  3. For ellipse 9x²+16y²=144, find the length of latus rectum.
  4. Tangent to x²/4+y²/3=1 at (1, 3/2): verify it's correct.

Unit Goal: AP/GP are straightforward marks in JEE — lock them down. Binomial and P&C require creative thinking; target the standard question types.

S1AP — nth Term, Sum, Applications1.5 hrs▼
Concepts
  • AP: aₙ = a + (n−1)d; Sₙ = n/2 × (2a + (n−1)d) = n/2 × (a+l)
  • Three numbers in AP: use a−d, a, a+d (symmetric form)
  • Sum of first n natural numbers: Σn = n(n+1)/2
  • Arithmetic mean: A = (a+b)/2
  • If Sₙ = an²+bn: first term = a+b, common difference = 2a
Tip: When three numbers are in AP, assume them as a-d, a, a+d. Two conditions → solve for a and d directly.
Practice Problems
  1. Find the 15th term and sum of first 20 terms of AP: 3, 7, 11, …
  2. Sum of first n terms = 3n² + 5n. Find the 10th term.
  3. If a₁, a₂, … aₙ are in AP and a₄/a₇ = 2/3, find a₆/a₈.
  4. Three numbers in AP with sum 21 and product 231. Find them.
S2GP — nth Term, Sum & Infinite GP1.5 hrs▼
Concepts
  • GP: aₙ = ar^(n−1); Sₙ = a(rⁿ−1)/(r−1) for r≠1
  • Three numbers in GP: use a/r, a, ar (symmetric form)
  • Infinite GP: S∞ = a/(1−r) valid only for |r| < 1
  • Geometric mean: G = √(ab)
  • Sum of n² and n³: Σn² = n(n+1)(2n+1)/6, Σn³ = [n(n+1)/2]²
Tip: Infinite GP formula S=a/(1−r) is only valid when |r|<1. Always check this condition before applying it.
Practice Problems
  1. GP: 2, 6, 18, … Find the 8th term and sum of first 6 terms.
  2. Find the sum to infinity: 1 + 1/3 + 1/9 + 1/27 + …
  3. If the sum of first 3 terms of a GP is 13 and sum of squares is 91, find the GP.
  4. Find 1² + 2² + 3² + … + 20².
S3Special Series & Binomial Theorem1.5 hrs▼
Concepts
  • Binomial: (a+b)ⁿ = Σ C(n,r) aⁿ⁻ʳ bʳ
  • General term: Tᵣ₊₁ = C(n,r) aⁿ⁻ʳ bʳ
  • Middle term: for (a+b)ⁿ, middle term at r = n/2
  • Term independent of x: set power of x = 0, solve for r
  • C(n,r) = n! / (r!(n−r)!) — Pascal's triangle
Tip: For "find coefficient of xᵏ", write the general term Tᵣ₊₁, express power of x in terms of r, set equal to k, solve for r.
Practice Problems
  1. Find the 5th term in (2x − 3/x)⁸.
  2. Find the term independent of x in (x + 1/x)¹⁰.
  3. Find the coefficient of x⁵ in (1+x)¹⁰.
  4. Using binomial, find (0.99)⁵ approximately.
S4Permutations & Combinations1.5 hrs▼
Concepts
  • P(n,r) = n!/(n−r)! — arrangements (order matters)
  • C(n,r) = n!/r!(n−r)! — selections (order doesn't matter)
  • Circular permutations: (n−1)! for n distinct objects
  • Identical objects: n! / (p! q! r! …) where p,q,r are repetition counts
  • Division into groups: multinomial coefficient
Tip: Ask: does ORDER matter? Yes → Permutation. No → Combination. This single question resolves most P&C problems.
Practice Problems
  1. How many 4-digit numbers can be formed from 1,2,3,4,5 (no repetition)?
  2. In how many ways can 10 students be seated in a circle?
  3. How many words (all letters) can be formed from MATHEMATICS?
  4. A committee of 3M, 2F from 5M and 4F. How many ways?
S5Probability1.5 hrs▼
Concepts
  • Classical probability: P(A) = favourable outcomes / total outcomes
  • P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
  • Mutually exclusive: P(A ∩ B) = 0
  • Independent events: P(A ∩ B) = P(A)·P(B)
  • Conditional: P(A|B) = P(A ∩ B)/P(B)
  • Bayes' theorem: P(A|B) = P(B|A)·P(A) / P(B)
Tip: Always verify: are events independent or mutually exclusive? These are different things. Independent means P(A∩B)=P(A)P(B). Mutually exclusive means P(A∩B)=0.
Practice Problems
  1. Two dice thrown. P(sum = 8)?
  2. P(A)=0.4, P(B)=0.5, P(A∩B)=0.2. Find P(A∪B) and P(A|B).
  3. Bag has 3R 4B balls. Two drawn without replacement. P(both same colour)?
  4. A speaks truth 4/5, B speaks truth 3/4. They agree on a fact. P(fact is true)?

D1Limits — Standard Forms & L'Hôpital's Rule1.5 hrs▼
Concepts
  • Standard limits: lim(sinx/x)=1, lim((1+x)^(1/x))=e, lim((aˣ−1)/x)=ln a as x→0
  • Indeterminate forms: 0/0, ∞/∞, 0·∞, ∞−∞, 1^∞, 0⁰, ∞⁰
  • L'Hôpital: for 0/0 or ∞/∞ forms, differentiate numerator and denominator separately
  • Squeeze (sandwich) theorem for special limits
  • Left and right limits; limit exists iff LHL = RHL
Tip: Before applying L'Hôpital, always verify the form is 0/0 or ∞/∞. Applying it to other forms is a common error.
Practice Problems

D2Continuity & Differentiability1.5 hrs▼
Concepts
  • Continuity at x=a: LHL = RHL = f(a) — all three must be equal
  • Differentiability: LHD = RHD at x=a
  • Differentiable ⟹ Continuous (converse not always true)
  • |x| is continuous but NOT differentiable at x=0
  • Piecewise functions: check at the join points separately
Tip: For piecewise functions, check continuity FIRST (limits from both sides = value). Only then check differentiability (derivatives from both sides).
Practice Problems

  2. Is f(x)=|x−2| differentiable at x=2? Justify.
  3. f(x) = x² (x<1), ax+b (x≥1). Find a, b for continuity and differentiability.
  4. Show f(x)=|x| is continuous everywhere but not differentiable at 0.
D3Derivatives — Product, Quotient, Chain Rule1.5 hrs▼
Concepts
  • d/dx (uv) = u'v + uv' (product rule)
  • d/dx (u/v) = (u'v − uv') / v² (quotient rule)
  • Chain rule: d/dx f(g(x)) = f'(g(x)) · g'(x)
  • d/dx(xⁿ) = nxⁿ⁻¹, d/dx(eˣ) = eˣ, d/dx(ln x) = 1/x
  • d/dx(sin x) = cos x, d/dx(cos x) = −sin x, d/dx(tan x) = sec²x
  • d/dx(arctan x) = 1/(1+x²), d/dx(arcsin x) = 1/√(1−x²)
Tip: For chain rule, identify the outer and inner functions. Differentiate the outer keeping the inner intact, then multiply by derivative of inner.
Practice Problems
  1. Differentiate: (x²+1)(x³−2x).
  2. Find dy/dx for y = sin(x²+3).
  3. Find dy/dx for y = x^x (use logarithmic differentiation).
  4. Differentiate: y = ln(sin x + cos x).
D4Implicit & Parametric Differentiation1.5 hrs▼
Concepts
  • Implicit: differentiate both sides w.r.t. x, collect dy/dx terms, solve
  • Parametric: x=f(t), y=g(t) → dy/dx = (dy/dt)/(dx/dt)
  • Second derivative parametrically: d²y/dx² = d/dt(dy/dx) / (dx/dt)
  • Logarithmic differentiation for products of powers: take ln both sides
Tip: For implicit differentiation, treat y as a function of x. When differentiating y², write 2y·(dy/dx) — the chain rule applies.
Practice Problems
  1. Find dy/dx if x² + y² = 25 (implicit).
  2. Find dy/dx for x=t², y=2t (parametric). Also find d²y/dx².
  3. If x^y = y^x, find dy/dx using logarithmic differentiation.
  4. Implicit: find dy/dx for sin(xy) = x.
D5Rolle's Theorem & Mean Value Theorem1.5 hrs▼
Concepts
  • Rolle's: f continuous on [a,b], differentiable on (a,b), f(a)=f(b) → ∃ c ∈ (a,b): f'(c)=0
  • Lagrange's MVT: f'(c) = (f(b)−f(a))/(b−a) for some c ∈ (a,b)
  • Geometrically: MVT says some tangent is parallel to the chord
  • Used to prove inequalities and existence of roots
Tip: Rolle's theorem guarantees a horizontal tangent between two equal-valued points. It's used to prove "at least one root" of f'(x)=0 exists.
Practice Problems
  1. Verify Rolle's theorem for f(x)=x²−5x+6 on [2,3].
  2. Apply LMVT to f(x)=x³ on [1,3]. Find c.
  3. Show that x³−3x+3=0 has exactly one real root greater than 1.
  4. Using MVT, show: |sin a − sin b| ≤ |a−b|.

A1Tangents & Normals1.5 hrs▼
Concepts
  • Slope of tangent at (x₁,y₁) = dy/dx evaluated at that point
  • Tangent: y − y₁ = m(x − x₁)
  • Normal: slope = −1/m; equation y − y₁ = (−1/m)(x − x₁)
  • Angle of intersection of two curves: angle between their tangents at the intersection point
  • Orthogonal curves: m₁m₂ = −1 at intersection
Tip: Always find the point of tangency first (substitute the given x into y). Then find dy/dx at that x. Then write the line.
Practice Problems
  1. Find equation of tangent and normal to y=x³−3x at (2, 2).
  2. Find point on y=x² where tangent has slope 4.
  3. Find angle between y=x² and y=x³ at their intersection point x=1.
  4. If normal to y=f(x) at (1,1) passes through (3,−1), find f'(1).
A2Maxima & Minima1.5 hrs▼
Concepts
  • Critical points: f'(x) = 0 or f'(x) undefined
  • First derivative test: f' changes + to − → local max; − to + → local min
  • Second derivative test: f''(c) < 0 → max; f''(c) > 0 → min
  • Global max/min on closed interval: check critical points AND endpoints
  • Optimisation: set up function, differentiate, set to zero, verify type
Tip: For optimisation problems, write the quantity to maximise/minimise as a function of ONE variable. Use any given constraint to eliminate extra variables first.
Practice Problems
  1. Find local max/min of f(x) = x³ − 12x + 5.
  2. A rectangle has perimeter 100 m. Find dimensions for maximum area.
  3. Find the maximum value of f(x) = x/(1+x²) for x>0.
  4. Find global max/min of f(x)=x³−3x on [0, 2].
A3Increasing, Decreasing & Concavity1.5 hrs▼
Concepts
  • Strictly increasing on interval: f'(x) > 0 throughout
  • Strictly decreasing: f'(x) < 0 throughout
  • Concave up: f''(x) > 0; concave down: f''(x) < 0
  • Inflection point: f'' changes sign
Tip: To find intervals of increase/decrease, solve f'(x) = 0, mark the roots on a number line, test signs of f' in each interval.
Practice Problems
  1. Find intervals where f(x)=x³−3x²−9x+2 is increasing.
  2. Find inflection points of f(x)=x⁴−6x².
  3. For f(x)=2x³−9x²+12x−5, find all local extrema and intervals of increase/decrease.
  4. Prove f(x)=sin x + tan x − 2x > 0 for x ∈ (0, π/2).
A4Rate of Change & Approximation1.5 hrs▼
Concepts
  • Rate of change: dy/dt = (dy/dx)(dx/dt) — chain rule for related rates
  • Linear approximation: f(x+δx) ≈ f(x) + f'(x)·δx
  • Percentage error: δy/y × 100 ≈ f'(x)/f(x) × δx × 100
  • Common rates: volume of sphere, surface area, water tank problems
Tip: In "rate of change" problems, always identify what's given (dx/dt or dr/dt) and what's asked (dy/dt or dV/dt). Connect them with the chain rule.
Practice Problems
  1. Sphere radius increasing at 2 cm/s. Rate of volume increase at r=5?
  2. Use approximation to find √(25.1) approximately.
  3. Ladder 10 m leans on wall. Bottom slides out at 1 m/s. Rate of top sliding when bottom is 6 m away?
  4. If edge of cube increases by 1%, find % increase in surface area.

I5Definite Integration — Properties & Shortcuts1.5 hrs▼

V1Vectors — Basics, Dot & Cross Products1.5 hrs▼
Concepts

Concepts
  • Types: square, identity, diagonal, symmetric (Aᵀ=A), skew-symmetric (Aᵀ=−A)
  • Matrix multiplication: (AB)ᵢⱼ = Σ AᵢₖBₖⱼ — not commutative in general
  • Determinant of 2×2: |ab;cd| = ad−bc
  • Sarrus rule for 3×3 determinant
  • Properties: |AB| = |A||B|; |Aᵀ| = |A|; |kA| = kⁿ|A|
  • Singular matrix: det = 0; not invertible
Tip: For 3×3 determinant, expand along the row or column with the most zeros. This minimises calculation.
Practice Problems
  1. For A=[[1,2],[3,4]] and B=[[5,6],[7,8]], find AB and BA. Are they equal?
  2. Find det of [[2,−1,3],[0,4,1],[5,0,−2]].
  3. If |A|=3, find |2A| for 3×3 matrix A. Also find |Aᵀ|.
  4. For what value of k is matrix [[2,3,k],[1,−1,2],[k,2,1]] singular?
V4Inverse Matrix & Linear Equations1.5 hrs▼
Concepts
  • Inverse: A⁻¹ = adj(A)/det(A), exists only when det(A) ≠ 0
  • Adjugate: transpose of cofactor matrix
  • AA⁻¹ = A⁻¹A = I (identity matrix)
  • Solving AX = B: X = A⁻¹B (Cramer's rule is alternative)
  • Cramer's rule: x = |D₁|/|D|, y = |D₂|/|D|, z = |D₃|/|D|
Tip: For 2×2, inverse is quick: swap diagonal, negate off-diagonal, divide by det. For 3×3, Cramer's rule is often faster than finding A⁻¹ in JEE.
Practice Problems
  1. Find inverse of A=[[2,3],[1,4]].
  2. Solve using Cramer's rule: 2x+3y=7, 3x−2y=1.
  3. If A²−5A+7I=0, show A⁻¹ = (5I−A)/7.
  4. Solve using matrices: x+y+z=6, 2x+y−z=1, x−2y+3z=7.
V5Mathematical Reasoning & Statistics1.5 hrs▼
Concepts
  • Mean = Σfᵢxᵢ / Σfᵢ; Variance σ² = Σfᵢ(xᵢ−x̄)² / Σfᵢ
  • Standard deviation = √Variance
  • Relation: Variance = Σfᵢxᵢ²/Σfᵢ − x̄²
  • Statement types: simple, compound; connectives: and(∧), or(∨), not(¬)
  • Contrapositive of "if p then q": "if not q then not p" — logically equivalent
Tip: Statistics in JEE is formulaic. Learn mean/variance formulas and apply systematically. 1-2 easy questions always appear.
Practice Problems
  1. Find mean and variance of: 2, 4, 4, 4, 5, 5, 7, 9.
  2. Write the contrapositive and converse of: "If it rains, the match is cancelled."
  3. If variance of 1, 2, 3, …, n is 10, find n.
  4. Identify tautology: (p ∧ q) → q.
