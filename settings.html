Back up your progress

Two ways to protect your data and access it from other devices:

📁
Save a backup file
Download a file to your computer — good for a quick one-off backup or moving to a new device.
☁
Sync to GitHub Not configured
Saves to a private file in your GitHub account. Once set up, one click to sync or restore from any device — no cables, no email, no USB.
1 Create a free GitHub account at github.com if you don't have one.
2 Go to github.com/settings/tokens → "Generate new token (classic)" → give it any name → tick only the gist checkbox → click Generate → copy the token.
3 Paste the token below, click Save Token, then click ↑ Sync Now.

Using a new device? Save the same token above → click "↓ Restore from GitHub" — it finds your backup automatically.
